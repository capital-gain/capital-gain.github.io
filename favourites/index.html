Учет инвестиций

  • Snowball - платный сервис для ведение учета сделок и полученных выплат, анализ состава портфеля и результатов, российские и иностранные ценные бумаги, импорт сделок из API или отчетов разных брокеров. Есть ограниченный бесплатный тариф. Данные хранятся на сервере, аккаунт лучше регистрировать анонимно, из брокерских отчетов перед импортом удалять личные данные.
  • IntelInvest - ещё один платный сервис для учета инвестиций с аналогичными возможностями. Есть ограниченный бесплатный тариф. Данные хранятся на сервере, аккаунт лучше регистрировать анонимно, из брокерских отчетов перед импортом удалять личные данные.
  • Portfolio Performance - бесплатное и open source десктопное приложение для учета инвестиций. Обладает широкими возможностями по ведению учета сделок и визуализации их результатов. Поддерживает любые инструменты, автоматическое или ручное обновление котировок, не хранит данные на сервере - ваши данные принадлежат только вам. Сложновато освоить для новичков и нет автоматического импорта сделок из популярных в России брокеров. По ссылке моя инструкция по настройке и источникам данных.
